# Laravel 12 + React 19 Full Stack Image Marketplace

**Author:** Muhammad Suleman  

---

## 🚀 Project Overview

This project is a **full-stack E-Commerce Image Marketplace** built using **Laravel 12+ and React 19**.  
It allows users to **buy, sell, and manage digital images** with secure authentication, payment integration, and a complete admin panel.

The platform simulates a real-world marketplace with modern features including user roles, image management, reviews, and financial transactions.

---

## 🧠 Key Objectives

- Build a production-level full-stack application  
- Implement scalable backend APIs using Laravel  
- Develop dynamic UI using React and Redux  
- Integrate real-world features like payments and role-based systems  

---

## 🛠 Tech Stack

### 🔹 Backend
- Laravel 12+ (REST API)
- Laravel Sanctum / JWT Authentication
- MySQL Database

### 🔹 Frontend
- React 19
- Redux Toolkit (State Management)
- Bootstrap 5

### 🔹 Tools & Services
- Stripe (Payment Integration)
- PayPal (Withdrawals)
- Cloud Storage (AWS S3 / Cloudinary)
- XAMPP (Local Development)
- VS Code

---

## 🔑 Core Features

### 👤 User Features
- Browse images with filters (category, price, extension)
- Search images by title
- View detailed image information
- Add to cart and wishlist
- User authentication (Register/Login)
- Role switching (Buyer ↔ Seller)
- Upload and manage images (Seller)
- Purchase images via Stripe
- Download purchased images
- Profile dashboard (Purchases, Images, Earnings, Withdrawals)
- Submit, update, and delete reviews

---

### 🛒 Marketplace Features
- Image preview with metadata (price, size, extension)
- Photographer details and download count
- Wishlist and cart system
- Checkout system with Stripe integration
- Order success handling

---

### 🛠 Admin Panel Features
- Manage categories (CRUD)
- Approve / Reject uploaded images
- Approve / Reject reviews
- Manage users (view/delete)
- View purchases and transactions
- Manage withdrawal requests
- Revenue system (70% user / 30% platform commission)
